Возможности программы

 1) Для создания вершины необходимо произвести двойной клик левой кнопкой мыши по месту, где должна быть создана вершина.

 2) Для переименования вершины необходимо произвести двойной клик по вершине, в открывшемся окне набрать новое имя вершины и подтвердить свой выбор

 3) Для изменения цвета вершины необходимо произвести по вершине клик правой кнопкой мыши, в открывшемся меню вершины выбрать соответствующее действие

 4) Для удаления вершины необходимо выбрать вершину левой кнопкой мыши, далее произвести по вершине клик правой кнопкой мыши и в открывшемся меню вершины выбрать соответствующее действие

 5) Для добавление ребра необходимо выбрать вершину, далее зажать клавишу Shift и выбрать вершину, в которой должно кончаться ребро

 6) Для удаления ребра необходимо выбрать ребро левой кнопкой мыши, далее правой кнопкой мыши вызвать меню ребра и выбрать соответствующее действие

 7) Для изменения веса ребра необходимо выбрать ребро левой кнопкой мыши, далее правой кнопкой мыши вызвать меню ребра и выбрать соответствующее действие

 8) Открытие таблицы смежности. Для того чтобы открыть сохраненную таблицу, необходимо нажать на кнопку "Файл -> Простой файл -> Открыть список" и выбрать текстовый файл. Пример такого файла - example.txt.

 9) Открытие форматированного графа. Для того чтобы открыть форматированный граф, необходимо нажать на кнопку "Файл -> Форматированный граф -> Открыть" и выбрать файл с расширением grph. Пример такого файла - example.grph. Данный формат файла сохраняет цвет, имена, расположение вершин, а также веса ребер.

 10) Открытие сжатого форматированного графа. Для того чтобы открыть сжатый форматированный граф, необходимо нажать на кнопку "Файл -> Новый форматированный граф -> Открыть" и выбрать файл с расширением zgrph. Пример такого файла - example.zgrph. Данный формат файла аналогичен grph, отличие - сжатие данных

 11) Копирование и вставка предназначены для вершин и ребер. Выделяем граф полностью (или его часть) мышью, нажимаем правую кнопку мыши на произвольной вершине выбранной области, "Копировать" (или в Меню -> Копировать, или Ctrl+C). Потом в произвольном пустом месте нажимаем правую кнопку мыши, "Вставить" (или в Меню -> Вставить, или Ctrl+V) и должно произойти дублирование выделенной области

 12) Максимальный поток. Для запуска поиска максимального потока в сети неоюходимо создать граф, нажать в меню "Максимальный поток -> Найти максимальный поток". В верхнем левом углу появится окно запуска алгоритма. Для выполнения алгоритма пошагово необходимо нажать правую стрелку (Сл. шаг). Для автоматического запуска поиска - левую зеленую стрелку (Запустить поиск).
